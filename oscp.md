## CLI Notes
Setup shell aliases for connecting to OVPN & Windows machine
Under `.bashrc` in Kali
```
cd
lab="openvpn YOUR_OVPN_FILE.ovpn"
win="rdesktop -u username -p password 1.2.3.4"
source .bashrc
```
